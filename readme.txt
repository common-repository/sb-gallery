Add Styling and hovers effects to default WordPresss Galleries

== Description ==

SB Gallery enhance the layout , styling and fancybox of your WordPress default gallery. 
Simply use WordPress default gallery and you will be able to add fancybox and other effects to your gallery in WordPress pages. 
No Extra efforts required. 

Features:

* Responsive Images
* Four Mouse over effects in the gallery images.
* Automatically add fancy box.
* Automatically scales large images to fit in window.
* Options to select the hover effect.
* Option activate/deactivate fancybox.

<a href="http://dev.responsiveidea.com/green_energy/wp/testing-gallery/" target="_blank">View demo</a> 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `sb_gallery` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a gallery in your WordPress pages / Post .
4. Simple Setting in the Settings -> Sb Gallery.


== Frequently Asked Questions ==

= Do i need to create the gallery sepertly ? =

You can just create the gallery as wordpress galleries it will automatically overwrites the default.

= Hover Effects are jquery based? =

Hover Effects are not jquery based they simply CSS3 Hovers.

= Does this support fancybox or colorbox? =

This plugin already have the fancbybox version 1.3 with it which can be activated or deactivate as per your choice.
